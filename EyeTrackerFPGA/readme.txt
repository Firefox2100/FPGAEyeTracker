To use the same code on different setups, after downloading the code gfrom github, go to the builds folder and delete the default.xml
Also, open the project and change the default setup control to the setup on which it will be run and rebuild the code

In the BufferToHost properties, general, select show all versions and select the FIFO - 64 Bit (v1.1.0 - legacy). Also make sure that NI1483 folder from z:\software\labview is copied to C:\Program Files (x86)\National Instruments\Shared\FlexRIO\IO Modules folder
